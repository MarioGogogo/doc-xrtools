# 校验相关

## platformOS

检查平台系统 返回 Andriod IOS PC

```jsx
import { platformOS } from 'xrTools';

platformOS();
```

## isJSON

判断是否是 json 对象

```jsx
import { isJSON } from 'xrTools';

isJSON(json);
```

## isPhone

判断是否是手机号码

```jsx
import { isPhone } from 'xrTools';

isPhone(phone);
```

## isCarNum

检验车牌号

```jsx
import { isCarNum } from 'xrTools';

isCarNum(num);
```

## isEmojiCharacter

判断输入框是否有表情

```jsx
import { isEmojiCharacter } from 'xrTools';

isEmojiCharacter(str);
```
